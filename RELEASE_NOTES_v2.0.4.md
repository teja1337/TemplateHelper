## Installation

### Option 1: Using Installer (Recommended)
1. Download `Helper_Installer.exe`
2. Run the installer
3. Follow on-screen instructions
4. Application will be installed to `Program Files\Helper`

### Option 2: Portable Executable
- Download `Helper.exe` for portable/standalone usage
- No installation required - run directly

### Updater
- `updater.exe` is included for automatic update checks and installations

## Files

- **Helper.exe** (32.6 MB) - Main application
- **updater.exe** (8.3 MB) - Update utility
- **Helper_Installer.exe** (40.6 MB) - Windows installer with uninstaller support

## System Requirements

- Windows 10 or later
- .NET Framework (included with Windows 10+)
- Python 3.14+ (for development only)

## Known Issues

- File Explorer icon display may require Windows icon cache refresh
  - Solution: Delete `%LocalAppData%\IconCache.db` and restart Explorer
